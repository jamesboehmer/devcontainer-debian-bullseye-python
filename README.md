# Debian Python Devcontainer

- Base image `mcr.microsoft.com/vscode/devcontainers/python:3.10-bullseye`
- User `vscode`
- Shell `/bin/zsh`
- Devcontainer features:
  - docker-in-docker
  - aws-cli
- VSCode extensions:
  - hashicorp.terraform
  - tamasfe.even-better-toml
  - samuelcolvin.jinjahtml
  - zainchen.json
  - redhat.vscode-yaml
- Poetry 2.0.1
- Tailscale (`tscale`) and Openvpn (`ovpn`) clients
- Google Cloud CLI
- ArgoCD CLI
- Auth0 CLI
- postgresql-14
- libpq-dev
- gcc
- default-jdk
- gh
- jq
- xdg-utils
- ffmpeg
- `ecr-login` script
